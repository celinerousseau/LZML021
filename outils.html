Analyse des données dans un fichier

Vous trouverez sur cette page différents outils d'analyse de texte.
Pour afficher ou cacher l'aide, cliquez sur le bouton "Aide".

Bienvenue sur l'aide !

Le bouton permet d'afficher... un outils surprise 👀

Pour commencer, choisissez le fichier .txt duquel vous désirez analyser les données en cliquant sur le bouton

Une alerte apparaîtra, vous proposant de segmenter le texte.
Cliquez sur "Ok" pour afficher et segmenter le texte.
Cliquez sur "Annuler" pour seulement afficher le texte.

Ne vous inquiétez pas, le bouton vous permettra de segmenter le texte si vous avez changé d'avis !

Les opérations peuvent prendre du temps selon la taille du texte ! Merci de patienter.

Le texte chargé sera affiché en bas à gauche, et les analyses en bas à droite.

Le bouton permet d'afficher le dictionnaire des mots les plus fréquents dans un tableau.
❗Attention ! Le Dictionnaire ne s'affiche que si le texte a été segmenté.

Le bouton vous permet de chercher une expression entrée dans "Pôle" et affichera les résultats dans leur contexte linéaire.
